# Product Store Website With Images

This is my fourth HTML assignment created as part of my Web and App Development learning journey at SMIT.

## About The Project

Product Store is a multi-page website built using HTML. 
This project contains different product categories with images and navigation between pages.

The purpose of this assignment was to practice HTML page structure, internal linking, folder organization, and adding images to webpages.

## Technologies Used

- HTML5

## Website Structure

- Home Page
- Shop Categories Page
- Fragrance Products Page
- General Items Page
- Groceries Page
- Medical Products Page

## Features

- Multi-page website navigation
- Product categories
- Product images display
- Organized folder structure
- Internal page linking
- Basic webpage content organization

## What I Learned

Through this assignment, I learned:

- How to add images in HTML
- How image paths work
- How to organize website folders
- How to create linked pages
- How to structure a complete multi-page website
